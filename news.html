CVE-2024-1068 Known exploited

A use-after-free vulnerability in the Linux kernel's nftt_tables component can be exploited to achieve local privilege escalation. The nftt_verdict_init() function allows positive values as drop error when hook verdict, and hence the nft_hook_slow() function can cause a double free vulnerability when NFT_DROP is issued with a drop error which resembles NF_ACCEPT. We recommend upgrading past commit 7542da46e32b03e9f6138b639f833a61bf1d6f60. Source: Google Inc.
